# Forca
Projeto desenvolvido em 2023 com o objetivo de implementar um jogo da forca em JavaScript, utilizando como temática nomes de marcas de carros. O projeto tem foco em lógica de programação, manipulação do DOM e interação por meio de eventos no navegador.

Visão geral

O jogo seleciona aleatoriamente uma marca automotiva e o usuário deve tentar adivinhar a palavra informando letras ao longo das tentativas. A interface exibe o progresso da palavra, os acertos e erros, além do estado atual do jogo.

Funcionalidades

Seleção aleatória de marcas de carros como palavras do jogo

Exibição progressiva das letras corretas

Registro de tentativas incorretas

Verificação automática de vitória ou derrota

Interação direta com o teclado ou botões na interface (conforme implementação)

Fluxo de uso

O jogo inicia com uma palavra oculta (marca de carro).

O usuário seleciona ou digita letras para tentar adivinhar.

As letras corretas são exibidas no campo da palavra.

Tentativas incorretas são contabilizadas até o limite permitido.

O jogo finaliza com mensagem de vitória ou derrota.
